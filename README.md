# Бот для Telegram-канала о ремонте/стройке/даче/дизайне

Полностью бесплатная связка: RSS → рерайт (Gemini) → картинка (Pollinations.ai) → пост в канал (Telegram Bot API) → запуск по расписанию (GitHub Actions).

## Что нужно сделать перед запуском

### 1. Создать Telegram-бота и получить токен
1. Написать `@BotFather` в Telegram → `/newbot` → следовать инструкциям.
2. Скопировать токен вида `123456:AAExxxxxxx` — это `TELEGRAM_BOT_TOKEN`.

### 2. Добавить бота в канал
1. Создать канал (если ещё не создан).
2. Добавить бота в администраторы канала (права на публикацию постов).
3. `TELEGRAM_CHANNEL_ID` — это либо юзернейм канала в формате `@my_channel` (если канал публичный), либо числовой ID (для приватных каналов) — его можно получить, переслав любое сообщение из канала боту `@userinfobot` или через `getUpdates`.

### 3. Получить бесплатный Groq API-ключ
1. Зайти на https://console.groq.com/keys (вход через Google/GitHub, без карты).
2. Нажать Create API Key, скопировать ключ.
3. Это `GROQ_API_KEY`.

### 4. Выложить код в GitHub-репозиторий
1. Создать новый репозиторий (можно приватный).
2. Загрузить туда все файлы из этой папки (`bot.py`, `requirements.txt`, `.github/workflows/post.yml`).

### 5. Добавить секреты в репозиторий
В репозитории: `Settings → Secrets and variables → Actions → New repository secret`, добавить три секрета:
- `TELEGRAM_BOT_TOKEN`
- `TELEGRAM_CHANNEL_ID`
- `GROQ_API_KEY`

### 6. Проверить вручную
`Actions → Автопостинг в канал → Run workflow` — запустит бота прямо сейчас, не дожидаясь расписания. Проверить, что пост появился в канале.

После этого workflow будет запускаться сам каждые 4 часа (можно поменять в `post.yml`, строка `cron`).

## Как добавить новые источники статей

Открыть `bot.py`, добавить новую ссылку на RSS-фид в список `SOURCES`. Больше ничего менять не нужно.

## Настройки, которые можно поменять

- `MAX_POSTS_PER_RUN` в `bot.py` — сколько постов публиковать за один запуск (сейчас 1 — расписание само даёт 3 запуска в день, так посты не выходят пачкой и разнесены по времени).
- Расписание — три строки `cron` в `post.yml`: 08:00, 13:00, 19:00 по московскому времени (утро/обед/вечер — пики активности в ленте). Время в файле указано в UTC (МСК минус 3 часа), поменять — просто отредактировать часы.
- Промпт для рерайта и для картинки — функции `rewrite_article` и `generate_image` в `bot.py`.

## Стоимость

Всё бесплатно на старте:
- Groq API — бесплатный лимит 14 400 запросов в день и 30 в минуту для модели Llama 3.1 8B (для одного канала это с большим запасом)
- Pollinations.ai — без ключа, без оплаты
- GitHub Actions — бесплатные минуты для приватных репозиториев (обычно с запасом хватает на такой график)
- Telegram Bot API — бесплатно всегда
